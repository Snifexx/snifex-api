\doxysubsection{Detailed Description}
General type hashmaps. 

This implementation of hashmap is somewhat reminiscent of stb-\/db\textquotesingle{}s one. The inner workings are the same\+: an hashmap that uses open addressing (or closed hashing) where buckets do not cointain the entries them self but contain an index to a vector where we store entries contigiously. Of course, since they mimic the generic-\/like feeling of my \doxylink{group__vector_ga21ec81b33f2746d76b94b77c7f990379}{Vec}, we have a different way of implementing parametric polymorphism, but the core hashmap implementation details are pretty much the same. I also implemented a default very simplistic hashing function, but added a way for users to use a custom hashing algorithm, and I highly suggest you do so.

All examples are \href{https://github.com/Snifexx/snifex-api/tree/docs/src/examples-and-tests}{\texttt{ here}}
