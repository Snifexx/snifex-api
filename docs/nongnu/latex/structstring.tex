\doxysubsection{Detailed Description}
A sized string view. 

This a simple sized string view... Nothing else. For the uninitiated\+:
\begin{DoxyItemize}
\item \textquotesingle{}sized\textquotesingle{} means that we carry the length of the data with the data (the char\texorpdfstring{$\ast$}{*})
\item \textquotesingle{}view\textquotesingle{} because the memory is not owned. That means that it is the user responsibility to manage the memory the string view is pointing to. The recommended way is using an arena.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
These strings are NOT zero tailing (since they\textquotesingle{}re string-\/views anyways)
