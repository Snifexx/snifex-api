\doxysubsection{Detailed Description}
A fixed-\/sized arena. 

A fixed-\/sized arena, that allocates a certain amount of memory when initialized and never gets reallocated again. Because of this allocations can fail when there is not enough memory allocated. Since reallocations never happen, \doxylink{group__arena_gafa4a34eff8578b37c7fa9a3e12188959}{arena\+\_\+alloc} return pointers, which is VERY useful for strings or scratch arenas
